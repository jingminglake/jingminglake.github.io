### 一、吉他

1. 爬格子水平：

手指 -> 速度 BPM（食指、中指）， 速度 BPM（中指、无名指），速度 BPM（拨片）

（1）4分音符
 
 1234 -> 240, 180, 110
 
 4321 -> 220, 180, 110

 1434243 -> 230, 175, 0
 
 1423 -> 180, 170, 0
 
 1324 -> 170, 175, 0
 
 4231 -> 170, 170, 0
 
 MI型音阶 -> 165, 160, 0

 
（2）8分音符

 1234 -> 105, 85, 0

（3）16分音符

1234 -> 55, 0, 0

4321 -> 45, 0, 0
 

2. 和弦水平：

F和弦 低音区杂音（无名指、食指无力）

C和弦 有时候中指、无名指按不到位

Am和弦 直接按 10%概率按不到

G和弦 无名指按不准，弯曲度不够
